Assignment — Class Example (To-Do App)

Welcome to the ClassExample folder!
This contains a simple To-Do App used to demonstrate converting a basic HTML/JS project into an Ionic-powered version.

Tech Used

HTML (basic structure)
JavaScript (DOM manipulation)
Ionic (CDN version — optional upgrade)

Files

index.html — Main To-Do App page
script.js — Contains addTask() logic
Ionic_Git_Template.zip
README.md 

Features

Add tasks using plain input/button
Upgradable to <ion-input> + <ion-button>
Dynamic list generation (<ul> or <ion-list>)

Instructions to Run Locally 
Just open index.html in your browser

Upgrade to Ionic (for Students)

To enable Ionic:

Uncomment the Ionic CDN links in index.html

Replace:

<input> → <ion-input>

<button> → <ion-button>

<ul> → <ion-list>

Replace script.js with the async version included in comments
